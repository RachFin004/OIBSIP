# Exploratory Data Analysis (EDA) on Retail Sales Dataset

## 📌 Oasis Infobyte Internship

### 👤 Author
**Rachaita Sarkar**

### 🎯 Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing patterns, sales trends, product performance, and generate actionable business insights using Python.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

- Dataset Inspection
- Data Cleaning & Validation
- Missing Value Analysis
- Statistical Summary
- Monthly Sales Trend Analysis
- Quarterly Sales Trend Analysis
- Customer Demographics Analysis
- Product Category Analysis
- Revenue Analysis by Product Category
- Revenue Analysis by Gender
- Correlation Heatmap
- Business Recommendations

---

## 📁 Dataset

**Retail Sales Dataset**

The dataset contains information such as:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity Purchased
- Price per Unit
- Total Amount

---

## 💡 Key Insights

- Identified monthly and quarterly sales trends.
- Analyzed customer demographics based on age and gender.
- Determined the best-performing product categories.
- Compared revenue generated across different customer groups.
- Used correlation analysis to understand relationships between numerical variables.
- Provided business recommendations based on the findings.

---

## ▶️ How to Run

1. Download the project files.
2. Open the Jupyter Notebook (`Retail_Sales_EDA.ipynb`).
3. Ensure `retail_sales_dataset.csv` is in the same folder.
4. Run all notebook cells.
